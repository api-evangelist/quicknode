# QuickNode (quicknode)

QuickNode is a blockchain infrastructure platform offering RPC, REST, and gRPC APIs across 77+ blockchains, plus services for IPFS pinning, streams of real-time blockchain data, webhooks, and a managed key-value store. The platform is positioned at developers building Web3 and onchain applications who need low-latency, reliable infrastructure without operating their own nodes.

### QuickNode Core API
JSON-RPC, REST, and gRPC endpoints for 77+ blockchains including Ethereum, Solana, Base, and Bitcoin. Each customer endpoint is provisioned at a unique URL of the form `https://{endpoint}.quiknode.pro/{token}/`.
